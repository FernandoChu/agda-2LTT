Idea

Given a pullback square

         f'
    C -------> B
    | ⌟        |
  g'|          | g
    ∨          ∨
    A -------> X
         f

then the exponentiated square given by postcomposition

                f' ∘ -
      (T → C) ---------> (T → B)
         |                  |
  g' ∘ - |                  | g ∘ -
         |                  |
         ∨                  ∨
      (T → A) ---------> (T → X)
                f ∘ -

is a pullback square for any type T.

Properties

  abstract

      is-equiv-tot-is-fiberwise-equivᵉ

  compute-standard-pullback-postcompᵉ :
    standard-pullbackᵉ (postcompᵉ Tᵉ fᵉ) (postcompᵉ Tᵉ gᵉ) ≃ᵉ coneᵉ fᵉ gᵉ Tᵉ
