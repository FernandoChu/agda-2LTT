Equivalence on total spaces induced by embedding on the base types

We saw above that given an embedding f : A ‚Ü™ B and a type family C over B we obtain an embedding

  Œ£ A (C ‚àò f) ‚Ü™ Œ£ B C.

This embedding can be upgraded to an equivalence if we furthermore know that the support of C is contained in the image of f. More precisely, if we are given a section ((b , c) : Œ£ B C) ‚Üí fiber f b, then it follows that

  Œ£ A (C ‚àò f) ‚âÉ Œ£ B C.

module _
  {l1 l2 l3 : Level} {A : UU l1} {B : UU l2} {C : B ‚Üí UU l3} (f : A ‚Ü™ B)
  (H : ((b , c) : Œ£ B C) ‚Üí fiber (map-emb f) b)
  where

  map-inv-Œ£-emb-base : Œ£ B C ‚Üí Œ£ A (C ‚àò map-emb f)
  pr1 (map-inv-Œ£-emb-base u) = pr1 (H u)
  pr2 (map-inv-Œ£-emb-base u) = inv-tr C (pr2 (H u)) (pr2 u)

  is-section-map-inv-Œ£-emb-base :
    is-section (map-Œ£-map-base (map-emb f) C) map-inv-Œ£-emb-base
  is-section-map-inv-Œ£-emb-base (b , c) =
    ap
      ( Œª s ‚Üí (pr1 s , inv-tr C (pr2 s) c))
      ( eq-is-contr (is-torsorial-Id' b))

  is-retraction-map-inv-Œ£-emb-base :
    is-retraction (map-Œ£-map-base (map-emb f) C) map-inv-Œ£-emb-base
  is-retraction-map-inv-Œ£-emb-base (a , c) =
    ap
      ( Œª s ‚Üí (pr1 s , inv-tr C (pr2 s) c))
      ( eq-is-prop (is-prop-map-is-emb (pr2 f) (map-emb f a)))

  equiv-Œ£-emb-base : Œ£ A (C ‚àò map-emb f) ‚âÉ Œ£ B C
  pr1 equiv-Œ£-emb-base = map-Œ£-map-base (map-emb f) C
  pr2 equiv-Œ£-emb-base =
    is-equiv-is-invertible
      map-inv-Œ£-emb-base
      is-section-map-inv-Œ£-emb-base
      is-retraction-map-inv-Œ£-emb-base
