## Idea

If `C : (x : A) ‚Üí B x ‚Üí ùí∞` is a type family, and `H : f ~ g` is a homotopy
between functions `f g : (x : A) ‚Üí B x`, then there is a natural transport
operation that transports an element `z : C x (f x)` along the homotopy `H` to
an element of type `C x (g x)`.
