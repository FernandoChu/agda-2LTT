## Idea

An identification `(pair x y) ＝ (pair x' y')` in a dependent pair type `Σ A B`
is equivalently described as a pair `pair α β` consisting of an identification
`α : x ＝ x'` and an identification `β : (tr B α y) ＝ y'`.
