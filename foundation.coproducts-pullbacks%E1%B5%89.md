## Idea

Given two
[commuting squares of maps](foundation-core.commuting-squares-of-maps.md),

```text
    C ------> B                  C' -----> B'
    |         |                  |         |
    |         |  g     and       |         | g'
    ∨         ∨                  ∨         ∨
    A ------> X                  A' -----> X',
         f                            f'
```

then their coproduct

```text
  C + C' ----> B + B'
    |            |
    |            | g + g'
    ∨            ∨
  A + A' ----> X + X'
         f + f'
```

is a [pullback](foundation-core.pullbacks.md) if each summand is.
