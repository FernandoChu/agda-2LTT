Idea

A {{#concept "complete precategory" Agda=is-complete-Precategory}} is a precategory that has all limits for diagrams from a specified universe.

More precisely, we say that a precategory D is (l1 , l2)-complete if for any C : Precategory l1 l2 and any functor F : C → D the type of limits of F is inhabited.

Definition

is-complete-Precategoryᵉ :
  (l1ᵉ l2ᵉ : Level) {l3ᵉ l4ᵉ : Level}
  (Dᵉ : Precategoryᵉ l3ᵉ l4ᵉ) →
  UUᵉ (lsuc l1ᵉ ⊔ lsuc l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ)
is-complete-Precategoryᵉ l1ᵉ l2ᵉ Dᵉ =
  (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ) →
  limit-Precategoryᵉ Cᵉ Dᵉ Fᵉ
