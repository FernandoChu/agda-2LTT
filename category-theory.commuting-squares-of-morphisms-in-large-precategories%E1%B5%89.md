## Idea

A square of morphisms

```text
  x ------> y
  |         |
  |         |
  ∨         ∨
  z ------> w
```

in a [large precategory](category-theory.large-precategories.md) `C` is said to
**commute** if there is an [identification](foundation-core.identity-types.md)
between both composites.
