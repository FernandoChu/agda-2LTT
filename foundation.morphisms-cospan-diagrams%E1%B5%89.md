## Idea

A
{{#concept "morphism of cospan diagrams" Disambiguation="of types" Agda=hom-cospan-diagram}}
is a commuting diagram of the form

```text
  A -----> X <----- B
  |        |        |
  |        |        |
  ∨        ∨        ∨
  A' ----> X' <---- B'.
```
