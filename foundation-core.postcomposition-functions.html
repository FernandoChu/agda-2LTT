Idea

Given a map f : X → Y and a type A, the {{#concept "postcomposition function" Agda=postcomp}}

  f ∘ - : (A → X) → (A → Y)

is defined by λ h x → f (h x).

Definitions

module _
  {l1 l2 l3 : Level} (A : UU l1) {X : UU l2} {Y : UU l3}
  where

  postcomp : (X → Y) → (A → X) → (A → Y)
  postcomp f = postcomp-Π A f
