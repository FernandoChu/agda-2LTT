Idea

A type family B over A is said to be {{#concept "univalent" Disambiguation="type family" Agda=is-univalent}} if the map

  equiv-tr B : x Ôºù y ‚Üí B x ‚âÉ B y

is an equivalence for every x y : A. By the univalence axiom, this is equivalent to the type family B being an embedding considered as a map.

Properties

The univalence axiom states that the identity family id : ùí∞ ‚Üí ùí∞ is univalent

is-univalent-UU :
  (l : Level) ‚Üí is-univalent (id {A = UU l})
is-univalent-UU l = univalence

Assuming the univalence axiom, type families are univalent if and only if they are embeddings as maps

Proof: We have the commuting triangle of maps

                ap B
       (x Ôºù y) -----> (B x Ôºù B y)
           \               /
            \             /
  equiv-tr B \           / equiv-eq
              ‚à®         ‚à®
              (B x ‚âÉ B y)

where the right edge is an equivalence by the univalence axiom. Hence, the top map is an equivalence if and only if the left map is.

  {l1 l2 : Level} {A : UU l1} {B : A ‚Üí UU l2}
  where

  abstract
    is-emb-is-univalent :
      is-univalent B ‚Üí is-emb B
    is-emb-is-univalent U x y =
      is-equiv-top-map-triangle
        ( equiv-tr B)
        ( equiv-eq)
        ( ap B)
        ( Œª where refl ‚Üí refl)
        ( univalence (B x) (B y))

      is-emb B ‚Üí is-univalent B
    is-univalent-is-emb is-emb-B x y =

        ( equiv-tr B)
        ( equiv-eq)
        ( ap B)
        ( Œª where refl ‚Üí refl)
        ( is-emb-B x y)
