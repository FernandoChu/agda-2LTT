Idea

A {{#concept "right Kan extension" Disambiguation="of a functor between precategories" Agda=is-right-kan-extension-Precategory}} of functor F : C → D between precategories along another functor p : C → C' is the universal right extension of F along p.

More concretely, we require the function right-extension-map-Precategory to be an equivalence.

Definition

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ l5ᵉ l6ᵉ : Level}
  (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (C'ᵉ : Precategoryᵉ l3ᵉ l4ᵉ) (Dᵉ : Precategoryᵉ l5ᵉ l6ᵉ)
  (pᵉ : functor-Precategoryᵉ Cᵉ C'ᵉ) (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  (Rᵉ : right-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ)
  where

  is-right-kan-extension-Precategoryᵉ :
    UUᵉ (l1ᵉ ⊔ l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ ⊔ l5ᵉ ⊔ l6ᵉ)
  is-right-kan-extension-Precategoryᵉ =
    ( Gᵉ : functor-Precategoryᵉ C'ᵉ Dᵉ) →
      is-equivᵉ (right-extension-map-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ Rᵉ Gᵉ)

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ l5ᵉ l6ᵉ : Level}
  (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (C'ᵉ : Precategoryᵉ l3ᵉ l4ᵉ) (Dᵉ : Precategoryᵉ l5ᵉ l6ᵉ)
  (pᵉ : functor-Precategoryᵉ Cᵉ C'ᵉ) (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  where

  right-kan-extension-Precategoryᵉ : UUᵉ (l1ᵉ ⊔ l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ ⊔ l5ᵉ ⊔ l6ᵉ)
  right-kan-extension-Precategoryᵉ =
    Σᵉ ( right-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ)
      ( is-right-kan-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ)

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ l5ᵉ l6ᵉ : Level}
  (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (C'ᵉ : Precategoryᵉ l3ᵉ l4ᵉ) (Dᵉ : Precategoryᵉ l5ᵉ l6ᵉ)
  (pᵉ : functor-Precategoryᵉ Cᵉ C'ᵉ) (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  (Rᵉ : right-kan-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ)
  where

  right-extension-right-kan-extension-Precategoryᵉ :
    right-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ
  right-extension-right-kan-extension-Precategoryᵉ = pr1ᵉ Rᵉ

  is-right-kan-extension-right-kan-extension-Precategoryᵉ :
    is-right-kan-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ (pr1ᵉ Rᵉ)
  is-right-kan-extension-right-kan-extension-Precategoryᵉ = pr2ᵉ Rᵉ

  extension-right-kan-extension-Precategoryᵉ :
    functor-Precategoryᵉ C'ᵉ Dᵉ
  extension-right-kan-extension-Precategoryᵉ =
    extension-right-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ
      right-extension-right-kan-extension-Precategoryᵉ

  natural-transformation-right-kan-extension-Precategoryᵉ :
    natural-transformation-Precategoryᵉ Cᵉ Dᵉ
      ( comp-functor-Precategoryᵉ Cᵉ C'ᵉ Dᵉ
        ( extension-right-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ
          right-extension-right-kan-extension-Precategoryᵉ)
        ( pᵉ))
      ( Fᵉ)
  natural-transformation-right-kan-extension-Precategoryᵉ =
    natural-transformation-right-extension-Precategoryᵉ Cᵉ C'ᵉ Dᵉ pᵉ Fᵉ
      right-extension-right-kan-extension-Precategoryᵉ
