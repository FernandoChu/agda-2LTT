Definition

A map is said to be {{#concept "decidable" Disambiguation="map of types" Agda=is-decidable-map}} if its fibers are decidable types.

module _
  {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ}
  where

  is-decidable-mapᵉ : (Aᵉ → Bᵉ) → UUᵉ (l1ᵉ ⊔ l2ᵉ)
  is-decidable-mapᵉ fᵉ = (yᵉ : Bᵉ) → is-decidableᵉ (fiberᵉ fᵉ yᵉ)

Properties

Decidable maps are closed under homotopy

abstract
  is-decidable-map-htpyᵉ :
    {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} {fᵉ gᵉ : Aᵉ → Bᵉ} →
    fᵉ ~ᵉ gᵉ → is-decidable-mapᵉ gᵉ → is-decidable-mapᵉ fᵉ
  is-decidable-map-htpyᵉ Hᵉ Kᵉ bᵉ =
    is-decidable-equivᵉ
      ( equiv-totᵉ (λ aᵉ → equiv-concatᵉ (invᵉ (Hᵉ aᵉ)) bᵉ))
      ( Kᵉ bᵉ)

Left cancellation for decidable maps

If a composite g ∘ f is decidable and the left factor g is injective, then the right factor f is decidable.

module _
  {l1ᵉ l2ᵉ l3ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} {Cᵉ : UUᵉ l3ᵉ} {fᵉ : Aᵉ → Bᵉ} {gᵉ : Bᵉ → Cᵉ}
  where

  abstract
    is-decidable-map-right-factor'ᵉ :
      is-decidable-mapᵉ (gᵉ ∘ᵉ fᵉ) → is-injectiveᵉ gᵉ → is-decidable-mapᵉ fᵉ
    is-decidable-map-right-factor'ᵉ GFᵉ Gᵉ yᵉ with (GFᵉ (gᵉ yᵉ))
    ... | inlᵉ qᵉ = inlᵉ (pr1ᵉ qᵉ ,ᵉ Gᵉ (pr2ᵉ qᵉ))
    ... | inrᵉ qᵉ = inrᵉ (λ xᵉ → qᵉ ((pr1ᵉ xᵉ) ,ᵉ apᵉ gᵉ (pr2ᵉ xᵉ)))

Retracts into types with decidable equality are decidable

is-decidable-map-retractionᵉ :
  {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} → has-decidable-equalityᵉ Bᵉ →
  (iᵉ : Aᵉ → Bᵉ) → retractionᵉ iᵉ → is-decidable-mapᵉ iᵉ
is-decidable-map-retractionᵉ dᵉ iᵉ (rᵉ ,ᵉ Rᵉ) bᵉ =
  is-decidable-iffᵉ
    ( λ (pᵉ : iᵉ (rᵉ bᵉ) ＝ᵉ bᵉ) → rᵉ bᵉ ,ᵉ pᵉ)
    ( λ tᵉ → apᵉ (iᵉ ∘ᵉ rᵉ) (invᵉ (pr2ᵉ tᵉ)) ∙ᵉ (apᵉ iᵉ (Rᵉ (pr1ᵉ tᵉ)) ∙ᵉ pr2ᵉ tᵉ))
    ( dᵉ (iᵉ (rᵉ bᵉ)) bᵉ)

Any map out of the empty type is decidable

abstract
  is-decidable-map-ex-falsoᵉ :
    {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → is-decidable-mapᵉ (ex-falsoᵉ {lᵉ} {Xᵉ})
  is-decidable-map-ex-falsoᵉ xᵉ = inrᵉ pr1ᵉ

The identity map is decidable

abstract
  is-decidable-map-idᵉ :
    {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → is-decidable-mapᵉ (idᵉ {lᵉ} {Xᵉ})
  is-decidable-map-idᵉ yᵉ = inlᵉ (yᵉ ,ᵉ reflᵉ)

Equivalences are decidable maps

abstract
  is-decidable-map-is-equivᵉ :
    {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} {fᵉ : Aᵉ → Bᵉ} →
    is-equivᵉ fᵉ → is-decidable-mapᵉ fᵉ
  is-decidable-map-is-equivᵉ Hᵉ xᵉ = inlᵉ (centerᵉ (is-contr-map-is-equivᵉ Hᵉ xᵉ))

The map on total spaces induced by a family of decidable maps is decidable

module _
  {l1ᵉ l2ᵉ l3ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : Aᵉ → UUᵉ l2ᵉ} {Cᵉ : Aᵉ → UUᵉ l3ᵉ}
  where

  is-decidable-map-totᵉ :
    {fᵉ : (xᵉ : Aᵉ) → Bᵉ xᵉ → Cᵉ xᵉ} →
    ((xᵉ : Aᵉ) → is-decidable-mapᵉ (fᵉ xᵉ)) → is-decidable-mapᵉ (totᵉ fᵉ)
  is-decidable-map-totᵉ {fᵉ} Hᵉ xᵉ =
    is-decidable-equivᵉ (compute-fiber-totᵉ fᵉ xᵉ) (Hᵉ (pr1ᵉ xᵉ) (pr2ᵉ xᵉ))

The map on total spaces induced by a decidable map on the base is decidable

module _
  {l1ᵉ l2ᵉ l3ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} (Cᵉ : Bᵉ → UUᵉ l3ᵉ)
  where

  is-decidable-map-Σ-map-baseᵉ :
    {fᵉ : Aᵉ → Bᵉ} → is-decidable-mapᵉ fᵉ → is-decidable-mapᵉ (map-Σ-map-baseᵉ fᵉ Cᵉ)
  is-decidable-map-Σ-map-baseᵉ {fᵉ} Hᵉ xᵉ =
    is-decidable-equiv'ᵉ (compute-fiber-map-Σ-map-baseᵉ fᵉ Cᵉ xᵉ) (Hᵉ (pr1ᵉ xᵉ))

Products of decidable maps are decidable

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} {Cᵉ : UUᵉ l3ᵉ} {Dᵉ : UUᵉ l4ᵉ}
  where

  is-decidable-map-productᵉ :
    {fᵉ : Aᵉ → Bᵉ} {gᵉ : Cᵉ → Dᵉ} →
    is-decidable-mapᵉ fᵉ → is-decidable-mapᵉ gᵉ → is-decidable-mapᵉ (map-productᵉ fᵉ gᵉ)
  is-decidable-map-productᵉ {fᵉ} {gᵉ} Fᵉ Gᵉ xᵉ =
    is-decidable-equivᵉ
      ( compute-fiber-map-productᵉ fᵉ gᵉ xᵉ)
      ( is-decidable-productᵉ (Fᵉ (pr1ᵉ xᵉ)) (Gᵉ (pr2ᵉ xᵉ)))

Coproducts of decidable maps are decidable

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} {Cᵉ : UUᵉ l3ᵉ} {Dᵉ : UUᵉ l4ᵉ}
  where

  is-decidable-map-coproductᵉ :
    {fᵉ : Aᵉ → Bᵉ} {gᵉ : Cᵉ → Dᵉ} →
    is-decidable-mapᵉ fᵉ →
    is-decidable-mapᵉ gᵉ →
    is-decidable-mapᵉ (map-coproductᵉ fᵉ gᵉ)
  is-decidable-map-coproductᵉ {fᵉ} {gᵉ} Fᵉ Gᵉ (inlᵉ xᵉ) =
    is-decidable-equiv'ᵉ (compute-fiber-inl-map-coproductᵉ fᵉ gᵉ xᵉ) (Fᵉ xᵉ)
  is-decidable-map-coproductᵉ {fᵉ} {gᵉ} Fᵉ Gᵉ (inrᵉ yᵉ) =
    is-decidable-equiv'ᵉ (compute-fiber-inr-map-coproductᵉ fᵉ gᵉ yᵉ) (Gᵉ yᵉ)

Decidable maps are closed under base change

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} {Cᵉ : UUᵉ l3ᵉ} {Dᵉ : UUᵉ l4ᵉ}
  {fᵉ : Aᵉ → Bᵉ} {gᵉ : Cᵉ → Dᵉ}
  where

  is-decidable-map-base-changeᵉ :
    cartesian-hom-arrowᵉ gᵉ fᵉ → is-decidable-mapᵉ fᵉ → is-decidable-mapᵉ gᵉ
  is-decidable-map-base-changeᵉ αᵉ Fᵉ dᵉ =
    is-decidable-equivᵉ
      ( equiv-fibers-cartesian-hom-arrowᵉ gᵉ fᵉ αᵉ dᵉ)
      ( Fᵉ (map-codomain-cartesian-hom-arrowᵉ gᵉ fᵉ αᵉ dᵉ))

Decidable maps are closed under retracts of maps

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ} {Xᵉ : UUᵉ l3ᵉ} {Yᵉ : UUᵉ l4ᵉ}
  {fᵉ : Aᵉ → Bᵉ} {gᵉ : Xᵉ → Yᵉ}
  where

  is-decidable-retract-mapᵉ :
    fᵉ retract-of-mapᵉ gᵉ → is-decidable-mapᵉ gᵉ → is-decidable-mapᵉ fᵉ
  is-decidable-retract-mapᵉ Rᵉ Gᵉ xᵉ =
    is-decidable-retract-ofᵉ
      ( retract-fiber-retract-mapᵉ fᵉ gᵉ Rᵉ xᵉ)
      ( Gᵉ (map-codomain-inclusion-retract-mapᵉ fᵉ gᵉ Rᵉ xᵉ))
