Idea

A {{#concept "limit" Disambiguation="of a functor of precategories" Agda=limit-Precategory}} of a functor F between precategories is a limiting cone over F. That is, a cone τ such that cone-map-Precategory C D F τ d is an equivalence for all d : obj-Precategory D.

Equivalently, the limit of F is a right kan extension of F along the terminal functor into the terminal precategory.

We show that both definitions coincide, and we will use the definition using limiting cones as our official one.

If a limit exists, we call the vertex of the limiting cone the vertex of the limit.

Definition

Limiting cones

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (Dᵉ : Precategoryᵉ l3ᵉ l4ᵉ)
  (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  where

  is-limiting-cone-Precategoryᵉ :
    cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ → UUᵉ (l1ᵉ ⊔ l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ)
  is-limiting-cone-Precategoryᵉ τᵉ =
    (dᵉ : obj-Precategoryᵉ Dᵉ) →
    is-equivᵉ (cone-map-Precategoryᵉ Cᵉ Dᵉ Fᵉ τᵉ dᵉ)

  limit-Precategoryᵉ : UUᵉ (l1ᵉ ⊔ l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ)
  limit-Precategoryᵉ =
    Σᵉ ( cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ)
      ( is-limiting-cone-Precategoryᵉ)

  cone-limit-Precategoryᵉ :
    limit-Precategoryᵉ →
    cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ
  cone-limit-Precategoryᵉ = pr1ᵉ

  vertex-limit-Precategoryᵉ :
    limit-Precategoryᵉ →
    obj-Precategoryᵉ Dᵉ
  vertex-limit-Precategoryᵉ τᵉ =
    vertex-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ (cone-limit-Precategoryᵉ τᵉ)

  is-limiting-limit-Precategoryᵉ :
    (τᵉ : limit-Precategoryᵉ) →
    is-limiting-cone-Precategoryᵉ (cone-limit-Precategoryᵉ τᵉ)
  is-limiting-limit-Precategoryᵉ = pr2ᵉ

  hom-cone-limit-Precategoryᵉ :
    (τᵉ : limit-Precategoryᵉ) →
    (φᵉ : cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ) →
    hom-Precategoryᵉ Dᵉ
      ( vertex-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ φᵉ)
      ( vertex-limit-Precategoryᵉ τᵉ)
  hom-cone-limit-Precategoryᵉ τᵉ φᵉ =
    map-inv-is-equivᵉ
      ( is-limiting-limit-Precategoryᵉ τᵉ
        ( vertex-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ φᵉ))
      ( natural-transformation-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ φᵉ)

Limits through right kan extensions

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (Dᵉ : Precategoryᵉ l3ᵉ l4ᵉ)
  (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  where

  is-limit-right-extension-Precategoryᵉ :
    right-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ →
    UUᵉ (l1ᵉ ⊔ l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ)
  is-limit-right-extension-Precategoryᵉ =
    is-right-kan-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ

  limit-Precategory'ᵉ : UUᵉ (l1ᵉ ⊔ l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ)
  limit-Precategory'ᵉ =
    right-kan-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (Dᵉ : Precategoryᵉ l3ᵉ l4ᵉ)
  (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ) (Lᵉ : limit-Precategory'ᵉ Cᵉ Dᵉ Fᵉ)
  where

  right-extension-limit-Precategory'ᵉ :
    right-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ
  right-extension-limit-Precategory'ᵉ =
    right-extension-right-kan-extension-Precategoryᵉ
      Cᵉ terminal-Precategoryᵉ Dᵉ (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ Lᵉ

  extension-limit-Precategory'ᵉ :
    functor-Precategoryᵉ terminal-Precategoryᵉ Dᵉ
  extension-limit-Precategory'ᵉ =
    extension-right-kan-extension-Precategoryᵉ
      Cᵉ terminal-Precategoryᵉ Dᵉ (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ Lᵉ

Properties

Being a limit is a property

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (Dᵉ : Precategoryᵉ l3ᵉ l4ᵉ)
  (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  where

  is-prop-is-limiting-cone-Precategoryᵉ :
    (τᵉ : cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ) →
    is-propᵉ (is-limiting-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ τᵉ)
  is-prop-is-limiting-cone-Precategoryᵉ τᵉ =
    is-prop-Πᵉ λ φᵉ → is-property-is-equivᵉ _

  is-prop-is-limit-Precategory'ᵉ :
    ( Rᵉ : right-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ) →
    is-propᵉ (is-limit-right-extension-Precategoryᵉ Cᵉ Dᵉ Fᵉ Rᵉ)
  is-prop-is-limit-Precategory'ᵉ Rᵉ =
    is-prop-Πᵉ λ Kᵉ → is-property-is-equivᵉ _

Limiting cones are equivalent to limits

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (Dᵉ : Precategoryᵉ l3ᵉ l4ᵉ)
  (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  where

  equiv-is-right-kan-extension-is-limiting-Precategoryᵉ :
    (τᵉ : cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ) →
    is-limiting-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ τᵉ ≃ᵉ
      is-right-kan-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ
      (map-equivᵉ (equiv-right-extension-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ) τᵉ)
  equiv-is-right-kan-extension-is-limiting-Precategoryᵉ τᵉ =
    equiv-Πᵉ _
      ( equiv-point-Precategoryᵉ Dᵉ)
      ( λ xᵉ →
        equiv-iff-is-propᵉ
          ( is-property-is-equivᵉ _)
          ( is-property-is-equivᵉ _)
          ( λ eᵉ →
            is-equiv-left-factorᵉ
            ( induced-right-extension-mapᵉ xᵉ)
            ( natural-transformation-constant-functor-Precategoryᵉ
              terminal-Precategoryᵉ Dᵉ)
            ( trᵉ is-equivᵉ (invᵉ (lemmaᵉ τᵉ xᵉ)) eᵉ)
            ( is-equiv-natural-transformation-constant-functor-Precategoryᵉ
              Dᵉ _ _))
          ( λ eᵉ →
            trᵉ is-equivᵉ (lemmaᵉ τᵉ xᵉ)
              ( is-equiv-compᵉ
                ( induced-right-extension-mapᵉ xᵉ)
                ( natural-transformation-constant-functor-Precategoryᵉ
                  terminal-Precategoryᵉ Dᵉ)
                ( is-equiv-natural-transformation-constant-functor-Precategoryᵉ
                  Dᵉ _ _)
                ( eᵉ))))
    where
      induced-right-extension-mapᵉ = λ xᵉ →
        right-extension-map-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
          ( terminal-functor-Precategoryᵉ Cᵉ) ( Fᵉ)
          ( map-equivᵉ (equiv-right-extension-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ) τᵉ)
          ( constant-functor-Precategoryᵉ terminal-Precategoryᵉ Dᵉ xᵉ)
      lemmaᵉ :
        ( τᵉ : cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ)
        ( xᵉ : obj-Precategoryᵉ Dᵉ) →
          ( right-extension-map-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
            ( terminal-functor-Precategoryᵉ Cᵉ) Fᵉ
            ( map-equivᵉ (equiv-right-extension-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ) τᵉ)
            ( constant-functor-Precategoryᵉ terminal-Precategoryᵉ Dᵉ xᵉ)) ∘ᵉ
          ( natural-transformation-constant-functor-Precategoryᵉ
            terminal-Precategoryᵉ Dᵉ) ＝ᵉ
        ( cone-map-Precategoryᵉ Cᵉ Dᵉ Fᵉ τᵉ xᵉ)
      lemmaᵉ τᵉ xᵉ =
        eq-htpyᵉ λ fᵉ →
          eq-htpy-hom-family-natural-transformation-Precategoryᵉ Cᵉ Dᵉ
            ( comp-functor-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
              ( point-Precategoryᵉ Dᵉ xᵉ)
              ( terminal-functor-Precategoryᵉ Cᵉ))
            ( Fᵉ)
            ( _)
            ( _)
            ( λ gᵉ → reflᵉ)

  equiv-limit-limit'-Precategoryᵉ :
    limit-Precategoryᵉ Cᵉ Dᵉ Fᵉ ≃ᵉ limit-Precategory'ᵉ Cᵉ Dᵉ Fᵉ
  equiv-limit-limit'-Precategoryᵉ =
    equiv-Σᵉ
      ( is-right-kan-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
        ( terminal-functor-Precategoryᵉ Cᵉ) Fᵉ)
      ( equiv-right-extension-cone-Precategoryᵉ Cᵉ Dᵉ Fᵉ)
      ( λ τᵉ → equiv-is-right-kan-extension-is-limiting-Precategoryᵉ τᵉ)

  limit-limit'-Precategoryᵉ :
    limit-Precategoryᵉ Cᵉ Dᵉ Fᵉ → limit-Precategory'ᵉ Cᵉ Dᵉ Fᵉ
  limit-limit'-Precategoryᵉ =
    map-equivᵉ equiv-limit-limit'-Precategoryᵉ

  limit'-limit-Precategoryᵉ :
    limit-Precategory'ᵉ Cᵉ Dᵉ Fᵉ → limit-Precategoryᵉ Cᵉ Dᵉ Fᵉ
  limit'-limit-Precategoryᵉ =
    map-inv-equivᵉ equiv-limit-limit'-Precategoryᵉ
