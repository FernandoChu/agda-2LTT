Idea

The {{#concept "maybe monad" Disambiguation="on types" Agda=Maybe WD="option type" WDID=Q7099015}} is an operation on types that adds one point. This is used, for example, to define partial functions, where a partial function f : X ⇀ Y is a map f : X → Maybe Y.

The maybe monad is an example of a monad that is not a modality, since it is not idempotent.

Definitions

The maybe monad

Maybeᵉ : {lᵉ : Level} → UUᵉ lᵉ → UUᵉ lᵉ
Maybeᵉ Xᵉ = Xᵉ +ᵉ unitᵉ

unit-Maybeᵉ : {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → Xᵉ → Maybeᵉ Xᵉ
unit-Maybeᵉ = inlᵉ

exception-Maybeᵉ : {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → Maybeᵉ Xᵉ
exception-Maybeᵉ = inrᵉ starᵉ

extend-Maybeᵉ :
  {l1ᵉ l2ᵉ : Level} {Xᵉ : UUᵉ l1ᵉ} {Yᵉ : UUᵉ l2ᵉ} → (Xᵉ → Maybeᵉ Yᵉ) → Maybeᵉ Xᵉ → Maybeᵉ Yᵉ
extend-Maybeᵉ fᵉ (inlᵉ xᵉ) = fᵉ xᵉ
extend-Maybeᵉ fᵉ (inrᵉ *ᵉ) = inrᵉ *ᵉ

map-Maybeᵉ :
  {l1ᵉ l2ᵉ : Level} {Xᵉ : UUᵉ l1ᵉ} {Yᵉ : UUᵉ l2ᵉ} → (Xᵉ → Yᵉ) → Maybeᵉ Xᵉ → Maybeᵉ Yᵉ
map-Maybeᵉ fᵉ (inlᵉ xᵉ) = inlᵉ (fᵉ xᵉ)
map-Maybeᵉ fᵉ (inrᵉ *ᵉ) = inrᵉ *ᵉ

The inductive definition of the maybe monad

data Maybe'ᵉ {lᵉ : Level} (Xᵉ : UUᵉ lᵉ) : UUᵉ lᵉ where
  unit-Maybe'ᵉ : Xᵉ → Maybe'ᵉ Xᵉ
  exception-Maybe'ᵉ : Maybe'ᵉ Xᵉ

The predicate of being an exception

is-exception-Maybeᵉ : {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → Maybeᵉ Xᵉ → UUᵉ lᵉ
is-exception-Maybeᵉ {lᵉ} {Xᵉ} xᵉ = (xᵉ ＝ᵉ exception-Maybeᵉ)

is-not-exception-Maybeᵉ : {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → Maybeᵉ Xᵉ → UUᵉ lᵉ
is-not-exception-Maybeᵉ xᵉ = ¬ᵉ (is-exception-Maybeᵉ xᵉ)

The predicate of being a value

is-value-Maybeᵉ : {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → Maybeᵉ Xᵉ → UUᵉ lᵉ
is-value-Maybeᵉ {lᵉ} {Xᵉ} xᵉ = Σᵉ Xᵉ (λ yᵉ → inlᵉ yᵉ ＝ᵉ xᵉ)

value-is-value-Maybeᵉ :
  {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} (xᵉ : Maybeᵉ Xᵉ) → is-value-Maybeᵉ xᵉ → Xᵉ
value-is-value-Maybeᵉ xᵉ = pr1ᵉ

eq-is-value-Maybeᵉ :
  {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} (xᵉ : Maybeᵉ Xᵉ) (Hᵉ : is-value-Maybeᵉ xᵉ) →
  inlᵉ (value-is-value-Maybeᵉ xᵉ Hᵉ) ＝ᵉ xᵉ
eq-is-value-Maybeᵉ xᵉ Hᵉ = pr2ᵉ Hᵉ

Maybe structure on a type

maybe-structureᵉ : {l1ᵉ : Level} (Xᵉ : UUᵉ l1ᵉ) → UUᵉ (lsuc l1ᵉ)
maybe-structureᵉ {l1ᵉ} Xᵉ = Σᵉ (UUᵉ l1ᵉ) (λ Yᵉ → Maybeᵉ Yᵉ ≃ᵉ Xᵉ)

Properties

The unit of Maybe is an embedding

abstract
  is-emb-unit-Maybeᵉ : {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → is-embᵉ (unit-Maybeᵉ {Xᵉ = Xᵉ})
  is-emb-unit-Maybeᵉ {lᵉ} {Xᵉ} = is-emb-inlᵉ Xᵉ unitᵉ

emb-unit-Maybeᵉ : {lᵉ : Level} (Xᵉ : UUᵉ lᵉ) → Xᵉ ↪ᵉ Maybeᵉ Xᵉ
pr1ᵉ (emb-unit-Maybeᵉ Xᵉ) = unit-Maybeᵉ
pr2ᵉ (emb-unit-Maybeᵉ Xᵉ) = is-emb-unit-Maybeᵉ

abstract
  is-injective-unit-Maybeᵉ :
    {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} → is-injectiveᵉ (unit-Maybeᵉ {Xᵉ = Xᵉ})
  is-injective-unit-Maybeᵉ = is-injective-inlᵉ

Being an exception is decidable

is-decidable-is-exception-Maybeᵉ :
  {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} (xᵉ : Maybeᵉ Xᵉ) → is-decidableᵉ (is-exception-Maybeᵉ xᵉ)
is-decidable-is-exception-Maybeᵉ {lᵉ} {Xᵉ} (inlᵉ xᵉ) =
  inrᵉ (λ pᵉ → ex-falsoᵉ (is-empty-eq-coproduct-inl-inrᵉ xᵉ starᵉ pᵉ))
is-decidable-is-exception-Maybeᵉ (inrᵉ starᵉ) = inlᵉ reflᵉ

is-decidable-is-not-exception-Maybeᵉ :
  {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} (xᵉ : Maybeᵉ Xᵉ) → is-decidableᵉ (is-not-exception-Maybeᵉ xᵉ)
is-decidable-is-not-exception-Maybeᵉ xᵉ =
  is-decidable-negᵉ (is-decidable-is-exception-Maybeᵉ xᵉ)

The values of the unit of the Maybe monad are not exceptions

abstract
  is-not-exception-unit-Maybeᵉ :
    {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} (xᵉ : Xᵉ) → is-not-exception-Maybeᵉ (unit-Maybeᵉ xᵉ)
  is-not-exception-unit-Maybeᵉ {lᵉ} {Xᵉ} xᵉ ()

For any element of type Maybe X we can decide whether it is a value or an exception

decide-Maybeᵉ :
  {lᵉ : Level} {Xᵉ : UUᵉ lᵉ} (xᵉ : Maybeᵉ Xᵉ) → is-value-Maybeᵉ xᵉ +ᵉ is-exception-Maybeᵉ xᵉ
decide-Maybeᵉ (inlᵉ xᵉ) = inlᵉ (xᵉ ,ᵉ reflᵉ)
decide-Maybeᵉ (inrᵉ starᵉ) = inrᵉ reflᵉ

Values are not exceptions

abstract
  is-not-exception-is-value-Maybeᵉ :
    {l1ᵉ : Level} {Xᵉ : UUᵉ l1ᵉ} (xᵉ : Maybeᵉ Xᵉ) →
    is-value-Maybeᵉ xᵉ → is-not-exception-Maybeᵉ xᵉ
  is-not-exception-is-value-Maybeᵉ {l1ᵉ} {Xᵉ} .(inlᵉ xᵉ) (xᵉ ,ᵉ reflᵉ) =
    is-not-exception-unit-Maybeᵉ xᵉ

If a point is not an exception, then it is a value

is-value-is-not-exception-Maybeᵉ :
  {l1ᵉ : Level} {Xᵉ : UUᵉ l1ᵉ} (xᵉ : Maybeᵉ Xᵉ) →
  is-not-exception-Maybeᵉ xᵉ → is-value-Maybeᵉ xᵉ
is-value-is-not-exception-Maybeᵉ xᵉ Hᵉ =
  map-right-unit-law-coproduct-is-emptyᵉ
    ( is-value-Maybeᵉ xᵉ)
    ( is-exception-Maybeᵉ xᵉ)
    ( Hᵉ)
    ( decide-Maybeᵉ xᵉ)

value-is-not-exception-Maybeᵉ :
  {l1ᵉ : Level} {Xᵉ : UUᵉ l1ᵉ} (xᵉ : Maybeᵉ Xᵉ) → is-not-exception-Maybeᵉ xᵉ → Xᵉ
value-is-not-exception-Maybeᵉ xᵉ Hᵉ =
  value-is-value-Maybeᵉ xᵉ (is-value-is-not-exception-Maybeᵉ xᵉ Hᵉ)

eq-is-not-exception-Maybeᵉ :
  {l1ᵉ : Level} {Xᵉ : UUᵉ l1ᵉ} (xᵉ : Maybeᵉ Xᵉ) (Hᵉ : is-not-exception-Maybeᵉ xᵉ) →
  inlᵉ (value-is-not-exception-Maybeᵉ xᵉ Hᵉ) ＝ᵉ xᵉ
eq-is-not-exception-Maybeᵉ xᵉ Hᵉ =
  eq-is-value-Maybeᵉ xᵉ (is-value-is-not-exception-Maybeᵉ xᵉ Hᵉ)

The two definitions of Maybe are equivalent

module _
  {l1ᵉ l2ᵉ : Level} {Xᵉ : UUᵉ l1ᵉ}
  where

  map-Maybe-Maybe'ᵉ : Maybeᵉ Xᵉ → Maybe'ᵉ Xᵉ
  map-Maybe-Maybe'ᵉ (inlᵉ xᵉ) = unit-Maybe'ᵉ xᵉ
  map-Maybe-Maybe'ᵉ (inrᵉ _) = exception-Maybe'ᵉ

  map-inv-Maybe-Maybe'ᵉ : Maybe'ᵉ Xᵉ → Maybeᵉ Xᵉ
  map-inv-Maybe-Maybe'ᵉ (unit-Maybe'ᵉ xᵉ) = inlᵉ xᵉ
  map-inv-Maybe-Maybe'ᵉ exception-Maybe'ᵉ = inrᵉ starᵉ

  is-section-map-inv-Maybe-Maybe'ᵉ :
    is-sectionᵉ map-Maybe-Maybe'ᵉ map-inv-Maybe-Maybe'ᵉ
  is-section-map-inv-Maybe-Maybe'ᵉ (unit-Maybe'ᵉ xᵉ) = reflᵉ
  is-section-map-inv-Maybe-Maybe'ᵉ exception-Maybe'ᵉ = reflᵉ

  is-retraction-map-inv-Maybe-Maybe'ᵉ :
    is-retractionᵉ map-Maybe-Maybe'ᵉ map-inv-Maybe-Maybe'ᵉ
  is-retraction-map-inv-Maybe-Maybe'ᵉ (inlᵉ xᵉ) = reflᵉ
  is-retraction-map-inv-Maybe-Maybe'ᵉ (inrᵉ xᵉ) = reflᵉ

  is-equiv-map-Maybe-Maybe'ᵉ : is-equivᵉ map-Maybe-Maybe'ᵉ
  is-equiv-map-Maybe-Maybe'ᵉ =
    is-equiv-is-invertibleᵉ
      ( map-inv-Maybe-Maybe'ᵉ)
      ( is-section-map-inv-Maybe-Maybe'ᵉ)
      ( is-retraction-map-inv-Maybe-Maybe'ᵉ)

  equiv-Maybe-Maybe'ᵉ : Maybeᵉ Xᵉ ≃ᵉ Maybe'ᵉ Xᵉ
  equiv-Maybe-Maybe'ᵉ = (map-Maybe-Maybe'ᵉ ,ᵉ is-equiv-map-Maybe-Maybe'ᵉ)

The extension of surjective maps is surjective

module _
  {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ}
  where

  is-surjective-extend-is-surjective-Maybeᵉ :
    {fᵉ : Aᵉ → Maybeᵉ Bᵉ} → is-surjectiveᵉ fᵉ → is-surjectiveᵉ (extend-Maybeᵉ fᵉ)
  is-surjective-extend-is-surjective-Maybeᵉ {fᵉ} Fᵉ yᵉ =
    elim-existsᵉ
      ( exists-structure-Propᵉ (Maybeᵉ Aᵉ) (λ zᵉ → extend-Maybeᵉ fᵉ zᵉ ＝ᵉ yᵉ))
      ( λ xᵉ pᵉ → intro-existsᵉ (inlᵉ xᵉ) pᵉ)
      ( Fᵉ yᵉ)

The functorial action of Maybe preserves surjective maps

module _
  {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ}
  where

  is-surjective-map-is-surjective-Maybeᵉ :
    {fᵉ : Aᵉ → Bᵉ} → is-surjectiveᵉ fᵉ → is-surjectiveᵉ (map-Maybeᵉ fᵉ)
  is-surjective-map-is-surjective-Maybeᵉ {fᵉ} Fᵉ (inlᵉ yᵉ) =
    elim-existsᵉ
      ( exists-structure-Propᵉ (Maybeᵉ Aᵉ) (λ zᵉ → map-Maybeᵉ fᵉ zᵉ ＝ᵉ inlᵉ yᵉ))
      ( λ xᵉ pᵉ → intro-existsᵉ (inlᵉ xᵉ) (apᵉ inlᵉ pᵉ))
      ( Fᵉ yᵉ)
  is-surjective-map-is-surjective-Maybeᵉ Fᵉ (inrᵉ *ᵉ) = intro-existsᵉ (inrᵉ *ᵉ) reflᵉ

There is a surjection from Maybe A + Maybe B to Maybe (A + B)

module _
  {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ}
  where

  map-maybe-coproductᵉ : Maybeᵉ Aᵉ +ᵉ Maybeᵉ Bᵉ → Maybeᵉ (Aᵉ +ᵉ Bᵉ)
  map-maybe-coproductᵉ (inlᵉ (inlᵉ xᵉ)) = inlᵉ (inlᵉ xᵉ)
  map-maybe-coproductᵉ (inlᵉ (inrᵉ starᵉ)) = inrᵉ starᵉ
  map-maybe-coproductᵉ (inrᵉ (inlᵉ xᵉ)) = inlᵉ (inrᵉ xᵉ)
  map-maybe-coproductᵉ (inrᵉ (inrᵉ starᵉ)) = inrᵉ starᵉ

  is-surjective-map-maybe-coproductᵉ : is-surjectiveᵉ map-maybe-coproductᵉ
  is-surjective-map-maybe-coproductᵉ (inlᵉ (inlᵉ xᵉ)) =
    unit-trunc-Propᵉ ((inlᵉ (inlᵉ xᵉ)) ,ᵉ reflᵉ)
  is-surjective-map-maybe-coproductᵉ (inlᵉ (inrᵉ xᵉ)) =
    unit-trunc-Propᵉ ((inrᵉ (inlᵉ xᵉ)) ,ᵉ reflᵉ)
  is-surjective-map-maybe-coproductᵉ (inrᵉ starᵉ) =
    unit-trunc-Propᵉ ((inlᵉ (inrᵉ starᵉ)) ,ᵉ reflᵉ)

There is a surjection from Maybe A × Maybe B to Maybe (A × B)

module _
  {l1ᵉ l2ᵉ : Level} {Aᵉ : UUᵉ l1ᵉ} {Bᵉ : UUᵉ l2ᵉ}
  where

  map-maybe-productᵉ : Maybeᵉ Aᵉ ×ᵉ Maybeᵉ Bᵉ → Maybeᵉ (Aᵉ ×ᵉ Bᵉ)
  map-maybe-productᵉ (inlᵉ aᵉ ,ᵉ inlᵉ bᵉ) = inlᵉ (aᵉ ,ᵉ bᵉ)
  map-maybe-productᵉ (inlᵉ aᵉ ,ᵉ inrᵉ starᵉ) = inrᵉ starᵉ
  map-maybe-productᵉ (inrᵉ starᵉ ,ᵉ inlᵉ bᵉ) = inrᵉ starᵉ
  map-maybe-productᵉ (inrᵉ starᵉ ,ᵉ inrᵉ starᵉ) = inrᵉ starᵉ

  is-surjective-map-maybe-productᵉ : is-surjectiveᵉ map-maybe-productᵉ
  is-surjective-map-maybe-productᵉ (inlᵉ (aᵉ ,ᵉ bᵉ)) =
    unit-trunc-Propᵉ ((inlᵉ aᵉ ,ᵉ inlᵉ bᵉ) ,ᵉ reflᵉ)
  is-surjective-map-maybe-productᵉ (inrᵉ starᵉ) =
    unit-trunc-Propᵉ ((inrᵉ starᵉ ,ᵉ inrᵉ starᵉ) ,ᵉ reflᵉ)
