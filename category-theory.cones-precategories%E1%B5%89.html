Idea

A {{#concept "cone" Disambiguation="over a functor between precategories" Agda=cone-Precategory}} over a functor F between precategories is a natural transformation from a constant functor to F.

In this context, we usually think of (and refer to) the functor F as a diagram in its codomain, A cone over such diagram then corresponds to an element d, called the vertex of the cone, equipped with components d → F x satisfying the naturality condition.

For example, if F corresponds to the diagram F x → F y, then a cone over F corresponds to a commuting triangle as below.

      d
    /   \
   /     \
  ∨       ∨
 Fx ----> Fy

Equivalently, we can see a cone over F as a right extension of F along the terminal functor into the terminal precategory.

Definitions

The type of cones over a functor

module _
  {l1ᵉ l2ᵉ l3ᵉ l4ᵉ : Level} (Cᵉ : Precategoryᵉ l1ᵉ l2ᵉ) (Dᵉ : Precategoryᵉ l3ᵉ l4ᵉ)
  (Fᵉ : functor-Precategoryᵉ Cᵉ Dᵉ)
  where

  cone-Precategoryᵉ : UUᵉ (l1ᵉ ⊔ l2ᵉ ⊔ l3ᵉ ⊔ l4ᵉ)
  cone-Precategoryᵉ =
    Σᵉ ( obj-Precategoryᵉ Dᵉ)
      ( λ dᵉ →
        natural-transformation-Precategoryᵉ Cᵉ Dᵉ
          ( constant-functor-Precategoryᵉ Cᵉ Dᵉ dᵉ)
          ( Fᵉ))

  make-cone-Precategoryᵉ :
    ( dᵉ : obj-Precategoryᵉ Dᵉ)
    ( αᵉ :
      ( xᵉ : obj-Precategoryᵉ Cᵉ) →
      ( hom-Precategoryᵉ Dᵉ dᵉ (obj-functor-Precategoryᵉ Cᵉ Dᵉ Fᵉ xᵉ))) →
    ( {xᵉ yᵉ : obj-Precategoryᵉ Cᵉ} (fᵉ : hom-Precategoryᵉ Cᵉ xᵉ yᵉ) →
      coherence-triangle-hom-Precategoryᵉ Dᵉ
        ( αᵉ xᵉ)
        ( αᵉ yᵉ)
        ( hom-functor-Precategoryᵉ Cᵉ Dᵉ Fᵉ fᵉ)) →
    cone-Precategoryᵉ
  pr1ᵉ (make-cone-Precategoryᵉ dᵉ αᵉ pᵉ) = dᵉ
  pr1ᵉ (pr2ᵉ (make-cone-Precategoryᵉ dᵉ αᵉ pᵉ)) = αᵉ
  pr2ᵉ (pr2ᵉ (make-cone-Precategoryᵉ dᵉ αᵉ pᵉ)) fᵉ =
    invᵉ (pᵉ fᵉ) ∙ᵉ invᵉ (right-unit-law-comp-hom-Precategoryᵉ Dᵉ _)

  vertex-cone-Precategoryᵉ : cone-Precategoryᵉ → obj-Precategoryᵉ Dᵉ
  vertex-cone-Precategoryᵉ = pr1ᵉ

  vertex-functor-cone-Precategoryᵉ :
    cone-Precategoryᵉ → functor-Precategoryᵉ Cᵉ Dᵉ
  vertex-functor-cone-Precategoryᵉ αᵉ =
    constant-functor-Precategoryᵉ Cᵉ Dᵉ (vertex-cone-Precategoryᵉ αᵉ)

  natural-transformation-cone-Precategoryᵉ :
    (αᵉ : cone-Precategoryᵉ) →
    natural-transformation-Precategoryᵉ Cᵉ Dᵉ
      ( vertex-functor-cone-Precategoryᵉ αᵉ)
      ( Fᵉ)
  natural-transformation-cone-Precategoryᵉ = pr2ᵉ

  component-cone-Precategoryᵉ :
    (αᵉ : cone-Precategoryᵉ) (xᵉ : obj-Precategoryᵉ Cᵉ) →
    hom-Precategoryᵉ Dᵉ
      ( vertex-cone-Precategoryᵉ αᵉ)
      ( obj-functor-Precategoryᵉ Cᵉ Dᵉ Fᵉ xᵉ)
  component-cone-Precategoryᵉ αᵉ =
    hom-family-natural-transformation-Precategoryᵉ Cᵉ Dᵉ
      ( vertex-functor-cone-Precategoryᵉ αᵉ)
      ( Fᵉ)
      ( natural-transformation-cone-Precategoryᵉ αᵉ)

  naturality-cone-Precategoryᵉ :
    (αᵉ : cone-Precategoryᵉ) {xᵉ yᵉ : obj-Precategoryᵉ Cᵉ}
    (fᵉ : hom-Precategoryᵉ Cᵉ xᵉ yᵉ) →
    coherence-triangle-hom-Precategoryᵉ Dᵉ
      ( component-cone-Precategoryᵉ αᵉ xᵉ)
      ( component-cone-Precategoryᵉ αᵉ yᵉ)
      ( hom-functor-Precategoryᵉ Cᵉ Dᵉ Fᵉ fᵉ)
  naturality-cone-Precategoryᵉ αᵉ {xᵉ} {yᵉ} fᵉ =
    invᵉ (right-unit-law-comp-hom-Precategoryᵉ Dᵉ _) ∙ᵉ
    invᵉ
      ( naturality-natural-transformation-Precategoryᵉ Cᵉ Dᵉ
        ( vertex-functor-cone-Precategoryᵉ αᵉ)
        ( Fᵉ)
        ( natural-transformation-cone-Precategoryᵉ αᵉ)
        ( fᵉ))

Precomposing cones

  cone-map-Precategoryᵉ :
    (τᵉ : cone-Precategoryᵉ)
    (dᵉ : obj-Precategoryᵉ Dᵉ) →
    (hom-Precategoryᵉ Dᵉ dᵉ (vertex-cone-Precategoryᵉ τᵉ)) →
    natural-transformation-Precategoryᵉ Cᵉ Dᵉ
      ( constant-functor-Precategoryᵉ Cᵉ Dᵉ dᵉ)
      ( Fᵉ)
  pr1ᵉ (cone-map-Precategoryᵉ τᵉ dᵉ fᵉ) xᵉ =
    comp-hom-Precategoryᵉ Dᵉ (component-cone-Precategoryᵉ τᵉ xᵉ) fᵉ
  pr2ᵉ (cone-map-Precategoryᵉ τᵉ dᵉ fᵉ) hᵉ =
    invᵉ (associative-comp-hom-Precategoryᵉ Dᵉ _ _ _) ∙ᵉ
    apᵉ
      ( λ gᵉ → comp-hom-Precategoryᵉ Dᵉ gᵉ fᵉ)
      ( invᵉ (naturality-cone-Precategoryᵉ τᵉ hᵉ)) ∙ᵉ
    invᵉ (right-unit-law-comp-hom-Precategoryᵉ Dᵉ _)

Properties

Characterization of equality of cones over functors between precategories

  coherence-htpy-cone-Precategoryᵉ :
    (αᵉ βᵉ : cone-Precategoryᵉ) →
    (pᵉ : vertex-cone-Precategoryᵉ αᵉ ＝ᵉ vertex-cone-Precategoryᵉ βᵉ) →
    UUᵉ (l1ᵉ ⊔ l4ᵉ)
  coherence-htpy-cone-Precategoryᵉ αᵉ βᵉ pᵉ =
    (xᵉ : obj-Precategoryᵉ Cᵉ) →
    coherence-triangle-hom-Precategoryᵉ Dᵉ
      ( hom-eq-Precategoryᵉ Dᵉ
        ( vertex-cone-Precategoryᵉ αᵉ)
        ( vertex-cone-Precategoryᵉ βᵉ)
        ( pᵉ))
      ( component-cone-Precategoryᵉ αᵉ xᵉ)
      ( component-cone-Precategoryᵉ βᵉ xᵉ)

  htpy-cone-Precategoryᵉ :
    (αᵉ βᵉ : cone-Precategoryᵉ) →
    UUᵉ (l1ᵉ ⊔ l3ᵉ ⊔ l4ᵉ)
  htpy-cone-Precategoryᵉ αᵉ βᵉ =
    Σᵉ ( vertex-cone-Precategoryᵉ αᵉ ＝ᵉ vertex-cone-Precategoryᵉ βᵉ)
      ( coherence-htpy-cone-Precategoryᵉ αᵉ βᵉ)

  refl-htpy-cone-Precategoryᵉ :
    (αᵉ : cone-Precategoryᵉ) →
    htpy-cone-Precategoryᵉ αᵉ αᵉ
  pr1ᵉ (refl-htpy-cone-Precategoryᵉ αᵉ) = reflᵉ
  pr2ᵉ (refl-htpy-cone-Precategoryᵉ αᵉ) xᵉ =
    invᵉ (right-unit-law-comp-hom-Precategoryᵉ Dᵉ _)

  htpy-eq-cone-Precategoryᵉ :
    (αᵉ βᵉ : cone-Precategoryᵉ) →
    αᵉ ＝ᵉ βᵉ →
    htpy-cone-Precategoryᵉ αᵉ βᵉ
  htpy-eq-cone-Precategoryᵉ αᵉ .αᵉ reflᵉ = refl-htpy-cone-Precategoryᵉ αᵉ

  is-torsorial-htpy-cone-Precategoryᵉ :
    (αᵉ : cone-Precategoryᵉ) →
    is-torsorialᵉ (htpy-cone-Precategoryᵉ αᵉ)
  is-torsorial-htpy-cone-Precategoryᵉ αᵉ =
    is-torsorial-Eq-structureᵉ
      ( is-torsorial-Idᵉ (vertex-cone-Precategoryᵉ αᵉ))
      ( pairᵉ (vertex-cone-Precategoryᵉ αᵉ) reflᵉ)
      ( is-contr-equivᵉ
        ( Σᵉ
          ( natural-transformation-Precategoryᵉ Cᵉ Dᵉ
            ( constant-functor-Precategoryᵉ Cᵉ Dᵉ (vertex-cone-Precategoryᵉ αᵉ)) Fᵉ)
              (λ τᵉ → τᵉ ＝ᵉ (natural-transformation-cone-Precategoryᵉ αᵉ)))
        ( equiv-totᵉ
          ( λ τᵉ →
            inv-equivᵉ
              ( extensionality-natural-transformation-Precategoryᵉ Cᵉ Dᵉ
                ( constant-functor-Precategoryᵉ Cᵉ Dᵉ (vertex-cone-Precategoryᵉ αᵉ))
                ( Fᵉ) _ _) ∘eᵉ
            equiv-Π-equiv-familyᵉ
              ( λ xᵉ →
                equiv-invᵉ (component-cone-Precategoryᵉ αᵉ xᵉ) (pr1ᵉ τᵉ xᵉ) ∘eᵉ
                equiv-concat'ᵉ
                  ( component-cone-Precategoryᵉ αᵉ xᵉ)
                  ( right-unit-law-comp-hom-Precategoryᵉ Dᵉ _))))
        ( is-torsorial-Id'ᵉ (natural-transformation-cone-Precategoryᵉ αᵉ)))

  is-equiv-htpy-eq-cone-Precategoryᵉ :
    (αᵉ βᵉ : cone-Precategoryᵉ) →
    is-equivᵉ (htpy-eq-cone-Precategoryᵉ αᵉ βᵉ)
  is-equiv-htpy-eq-cone-Precategoryᵉ αᵉ =
    fundamental-theorem-idᵉ
      ( is-torsorial-htpy-cone-Precategoryᵉ αᵉ)
      ( htpy-eq-cone-Precategoryᵉ αᵉ)

  equiv-htpy-eq-cone-Precategoryᵉ :
    (αᵉ βᵉ : cone-Precategoryᵉ) →
    (αᵉ ＝ᵉ βᵉ) ≃ᵉ htpy-cone-Precategoryᵉ αᵉ βᵉ
  pr1ᵉ (equiv-htpy-eq-cone-Precategoryᵉ αᵉ βᵉ) = htpy-eq-cone-Precategoryᵉ αᵉ βᵉ
  pr2ᵉ (equiv-htpy-eq-cone-Precategoryᵉ αᵉ βᵉ) =
    is-equiv-htpy-eq-cone-Precategoryᵉ αᵉ βᵉ

  eq-htpy-cone-Precategoryᵉ :
    (αᵉ βᵉ : cone-Precategoryᵉ) →
    htpy-cone-Precategoryᵉ αᵉ βᵉ →
    αᵉ ＝ᵉ βᵉ
  eq-htpy-cone-Precategoryᵉ αᵉ βᵉ =
    map-inv-equivᵉ (equiv-htpy-eq-cone-Precategoryᵉ αᵉ βᵉ)

A cone is a right extension along the terminal map

  equiv-right-extension-cone-Precategoryᵉ :
    cone-Precategoryᵉ ≃ᵉ
    right-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ
  equiv-right-extension-cone-Precategoryᵉ =
    equiv-Σ-equiv-baseᵉ
    ( λ Kᵉ → natural-transformation-Precategoryᵉ Cᵉ Dᵉ
      ( comp-functor-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
        ( Kᵉ)
        ( terminal-functor-Precategoryᵉ Cᵉ))
      ( Fᵉ))
    ( equiv-point-Precategoryᵉ Dᵉ)

  right-extension-cone-Precategoryᵉ :
    cone-Precategoryᵉ →
    right-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ
  right-extension-cone-Precategoryᵉ =
    map-equivᵉ equiv-right-extension-cone-Precategoryᵉ

  cone-right-extension-Precategoryᵉ :
    right-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ →
    cone-Precategoryᵉ
  cone-right-extension-Precategoryᵉ =
    map-inv-equivᵉ equiv-right-extension-cone-Precategoryᵉ

  vertex-right-extension-Precategoryᵉ :
    right-extension-Precategoryᵉ Cᵉ terminal-Precategoryᵉ Dᵉ
      (terminal-functor-Precategoryᵉ Cᵉ) Fᵉ →
    obj-Precategoryᵉ Dᵉ
  vertex-right-extension-Precategoryᵉ Rᵉ =
    vertex-cone-Precategoryᵉ
      ( cone-right-extension-Precategoryᵉ Rᵉ)
