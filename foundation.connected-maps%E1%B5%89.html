A map that satisfies the dependent universal property for connected maps is a connected map

Proof: Consider a map f : A ‚Üí B such that the precomposition function

  - ‚àò f : ((b : B) ‚Üí P b) ‚Üí ((a : A) ‚Üí P (f a))

is an equivalence for every family P of k-truncated types. Then it follows that the precomposition function

  - ‚àò f : ((b : B) ‚Üí ‚ïëfiber f b‚ïë‚Çñ) ‚Üí ((a : A) ‚Üí ‚ïëfiber f (f a)‚ïë‚Çñ)

is an equivalence. In particular, the element Œª a ‚Üí Œ∑ (a , refl) in the codomain of this equivalence induces an element c b : ‚ïëfiber f b‚ïë‚Çñ for each b : B. We take these elements as our centers of contraction. Note that by construction, we have an identification c (f a) Ôºù Œ∑ (a , refl).

To construct a contraction of ‚ïëfiber f b‚ïë‚Çñ for each b : B, we have to show that

  (b : B) (u : ‚ïëfiber f b‚ïë‚Çñ) ‚Üí c b Ôºù u.

Since the type c b Ôºù u is k-truncated, this type is equivalent to the type (b : B) (u : fiber f b) ‚Üí c b Ôºù Œ∑ u. By reduction of the universal quantification over the fibers we see that this type is equivalent to the type

  (a : A) ‚Üí c (f a) Ôºù Œ∑ (a , refl).

This identification holds by construction of c.
