## Idea

Given functions `f : A → B` and `g : X → Y` the
{{#concept "bicomposition function" Disambiguation="types" Agda=bicomp}} is the
map

```text
  g ∘ - ∘ f : (B → X) → (A → Y)
```

defined by `λ h x → g (h (f x))`.
